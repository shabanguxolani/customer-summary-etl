# Customer Summary ETL Pipeline 🚀

A practice ETL pipeline built in SQLite as part of 
my 2 hours a day SQL learning journey.

## What It Does
- Extracts data from 3 raw tables (Customers, Orders, Shippings)
- Transforms data using CTEs and CASE WHEN logic
- Loads into a target summary table
- Performs data quality checks and NULL handling
